# دليل النشر والتشغيل - Anime Power Scales

## 🚀 متطلبات التشغيل

- Node.js 22+
- pnpm (أو npm/yarn)
- متصفح حديث (Chrome, Firefox, Safari, Edge)

## 📦 التثبيت والتشغيل

### 1. تثبيت المكتبات
```bash
cd /home/ubuntu/Anime-Power-Scale/artifacts/anime-battles
pnpm install
```

### 2. تشغيل خادم التطوير
```bash
PORT=5173 BASE_PATH="/" pnpm dev
```

### 3. الوصول إلى التطبيق
- **المحلي:** http://localhost:5173
- **الشبكة:** http://169.254.0.21:5173

## 🔧 متغيرات البيئة (اختيارية)

لتفعيل الميزات المتقدمة، أضف المتغيرات التالية إلى ملف `.env`:

```env
# Groq API (للبحث المتقدم)
VITE_GROQ_API_KEY=your_groq_api_key

# Google Gemini API (للتحليل المحسّن)
VITE_GOOGLE_API_KEY=your_google_api_key
```

## 📊 الميزات الرئيسية

### ✨ التصميم المحسّن
- شعار احترافي مع تأثيرات حماسية
- أيقونات جذابة للشخصيات
- خطوط واضحة وملونة في جميع الأقسام
- تدرجات لونية جذابة

### 🎮 محاكاة المواجهات
- نظام قتالي متقدم مع 10 جولات
- 5 أنواع من الإجراءات القتالية
- حساب الضرر الديناميكي
- سجل تفصيلي للمعارك

### 🔍 البحث المتقدم
- البحث في مصادر متعددة (MyAnimeList, Wiki, إلخ)
- دعم نماذج ذكاء اصطناعي متعددة
- عرض درجة الثقة لكل نتيجة
- روابط مباشرة للمصادر

### ✅ إصلاحات الأداء
- إصلاح مشاكل التمرير في القوائم
- تحسين استجابة الواجهة
- معالجة أفضل للأخطاء

## 🧪 الاختبار

### اختبار الميزات الأساسية
1. **الصفحة الرئيسية:** تحقق من ظهور الشعار الجديد
2. **قائمة الشخصيات:** تحقق من ظهور الأيقونات
3. **صفحة الشخصية:** اختبر البحث المتقدم
