## 🎯 خطوات النشر (سهلة جداً):

### الخطوة 1️⃣: إنشاء حساب Netlify (مجاني)
```
1. اذهب إلى: https://netlify.com
2. انقر على "Sign Up"
3. اختر GitHub أو Google
4. أكمل التسجيل
```

### الخطوة 2️⃣: إنشاء حساب Supabase (مجاني)
```
1. اذهب إلى: https://supabase.com
2. انقر على "Start your project"
3. اختر "Create a new project"
4. ملأ البيانات واختر كلمة مرور قوية
5. انتظر إنشاء المشروع (2-3 دقائق)
```

### الخطوة 3️⃣: إنشاء الجداول في Supabase
```
1. في Supabase، اذهب إلى "SQL Editor"
2. انسخ الكود من ملف: NETLIFY_DEPLOYMENT.md
3. نفذ الأوامر
```

### الخطوة 4️⃣: الحصول على مفاتيح API
```
1. في Supabase، اذهب إلى Settings → API
2. انسخ:
   - Project URL → VITE_SUPABASE_URL
   - anon public → VITE_SUPABASE_ANON_KEY
```

### الخطوة 5️⃣: ربط GitHub مع Netlify
```
1. في Netlify، انقر على "New site from Git"
2. اختر GitHub
3. اختر مستودع: Anime-Power-Scale
4. انقر على "Deploy site"
```

### الخطوة 6️⃣: إضافة متغيرات البيئة
```
1. في Netlify، اذهب إلى: Site settings → Build & deploy → Environment
2. أضف المتغيرات:
   - VITE_SUPABASE_URL = (من Supabase)
   - VITE_SUPABASE_ANON_KEY = (من Supabase)
```

### الخطوة 7️⃣: تفعيل النشر التلقائي
```
1. انقر على "Trigger deploy"
2. انتظر حتى ينتهي البناء (2-5 دقائق)
3. ستحصل على رابط مثل: https://anime-power-scales-xxxxx.netlify.app
```
